#### 색, 채도, 명도 

    - 채도 - 색을 뺏는다 (탁해보임/맑고)
    - 명도 - 밝고 어두움(빛) (무게감- )
    
### vivid - 원색 

### 파스텔 톤 - 가볍고 경쾌한

### 다크한 톤 - 무겁고 다크함

### saturation - 채도
     - 높이면 높일 수록 원색으로 가깝게 간다 
     - 낮추면 채도가 낮아 진다 
     
### Hue - 색

### lightness - 명도
